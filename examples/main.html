Marquee

Use the Marquee directive to slide content horizontally across the screen. Just like a really cool news alert!

Example

Use the <au-marquee> along with a speed and a direction attribute to get a scrolling element. The attribute take a number and a string respectively. If passing a literal value directly to direction, it should be done with single quote marks as such: direction="'right'".
