AngularJS directives to help create an AwesomeUI for your web page. Get ahead of your competitors instantly!

Marquee

Use the Marquee directive to slide content horizontally across the screen. Just like a really cool news alert!

Example

Use the <au-marquee> along with a speed and a direction attribute to get a scrolling element. The attribute takes a number and a string respectively. If passing a literal value directly to direction, it should be done with single quote marks as such: direction="'right'".

Default AU-Marquee
Fast AU-Marquee
Reverse AU-Marquee
Two AU-Marquees on the same lineAnother AU-Marquee on the same line
EXAMPLE
<au-marquee speed="3" direction="'right'">Default AU-Marquee</au-marquee><br/>
<au-marquee speed="9" direction="'right'">Fast AU-Marquee</au-marquee><br/>
<au-marquee speed="6" direction="'left'">Reverse AU-Marquee</au-marquee><br/>
<au-marquee speed="3" direction="'right'">Two AU-Marquees on the same line</au-marquee><au-marquee speed="3" direction="'left'">Another AU-Marquee on the same line</au-marquee>

Attributes

NameTypeValid InputDescription
speedNumberAny number.Higher numbers mean a greater speed at which the element will scroll.
directionString'left' or 'right' accepted only.The direction in which the element will scroll.

Spinner

Use the Spinner directive to spin content clockwise or anti-clockwise. Dreidel, dreidel, dreidel!

Example

Use the <au-spinner> along with a speed and a direction attribute to get a spinning element. The attributes take a number and a string respectively. If passing a literal value directly to direction, it should be done with single quote marks as such: direction="'clockwise'". You can even use images inside this directive.

Spinning one way! Spinning faster the other way!
EXAMPLE
<au-spinner speed="3" direction="'clockwise'">Spinning one way!</au-spinner>
<au-spinner speed="6" direction="'anti-clockwise'">Spinning faster the other way!</au-spinner>
<au-spinner speed="3" direction="'clockwise'"><img src="favicon.crush.png"/></au-spinner>

Attributes

NameTypeValid InputDescription
speedNumberAny number.Higher numbers mean a greater speed at which the element will scroll.
directionString'clockwise' or 'anti-clockwise' accepted only.The direction in which the element will scroll.

Seesaw

The Seesaw directive lets you teeter elements back and forth. It's like the end of the Italian Job!

Example

Use the <au-seesaw> along with a speed and a rotation-limit attribute to get a seesawing element. The attributes both take numbers. You can even use images inside this directive.

Whooooaaaaaa! Full circle! Cool! Slow double flip! Yeah!
EXAMPLE
<au-seesaw speed="3" rotation-limit="45">Whooooaaaaaa!</au-seesaw>
<au-seesaw speed="6" rotation-limit="180">Full circle! Cool!</au-seesaw>
<au-seesaw speed="1" rotation-limit="360">Slow double flip! Yeah!</au-seesaw>
<au-seesaw speed="3" rotation-limit="30"><img src="favicon.crush.png" /></au-seesaw>

Attributes

NameTypeValid InputDescription
speedNumberAny number.Higher numbers mean a greater speed at which the element will scroll.
rotation-limitNumberAny number.The limit the rotation will reach, both above and below zero. E.g.: 90 will stop the rotation at +90 as well as -90 degrees.
